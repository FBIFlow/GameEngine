Общий ресурс для взаимодействия между Producers и Executors.

  Получает от Producer:
    - запрос на вход/выход в очередь
    - запрос на создание/удаление ячейки очереди
    - запрос на открытие/закрытие ячейки очереди

  Отправляет на Producer:
    - данные о существующих сессиях(SessionHolders)
    - данные о обновлении статистики
    - ответ о наличии свободной сессии для ячейки очереди

  Получает от Executor:
    - данные о сессиях
    - ответ о создании сессии

  Отправляет на Executor:
    - запрос на создание сессии
